## Goto Selection

Sublime Text package that pre-fills the Goto Anything overlay with the
selected text or, if nothing is selected, with the word under the cursor.
The default keybinding is <kbd>SUPER</kbd>+<kbd>SHIFT</kbd>+<kbd>;</kbd>
(substitute <kbd>SUPER</kbd> for <kbd>CTRL</kbd> on windows/linux).

### Installation

Download _Goto Selection_ from [Package Control](https://packagecontrol.io/)
(or copy the files to a folder inside the _Packages/_ directory).
